preg_replace \e 模式（PHP 5.5 开始废弃，PHP 7 直接移除）

/e 修正符（modifier）是 PHP 的一个特性，使得 preg_replace() 将替换参数 replacement 当作 PHP 代码执行。在匹配到的字符串进行替换时，会先解析替换字符串，然后执行其中的 PHP 代码。
